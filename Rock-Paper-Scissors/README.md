# 🇹🇷 Türkçe

## 📌 Proje Hakkında

Bu proje Python ile geliştirilmiş basit bir **Taş Kağıt Makas** oyunudur.

Oyuncu bilgisayara karşı oynar. Bilgisayar rastgele bir seçim yapar ve oyun kurallarına göre kazanan belirlenir.

Oyunun amacı **3 puana ulaşarak oyunu kazanmaktır**.

---

## ✨ Özellikler

- 🎮 Oyuncu - bilgisayar karşılaşması
- 🤖 Rastgele bilgisayar seçimi
- 🏆 3 puana ulaşan taraf kazanır
- 📊 Skor görüntüleme sistemi
- ❌ Geçersiz giriş kontrolü
- 🚪 Oyundan çıkış seçeneği

---

## 🕹️ Kullanım

Oyunu çalıştırmak için:

```bash
python tas_kagit_makas.py
```

Oyun başladıktan sonra aşağıdaki seçeneklerden birini girin:

```
taş
kağıt
makas
```

Ek komutlar:

```
sc → Skoru gösterir
q  → Oyundan çıkar
```

# 🇬🇧 English

## 📌 About The Project

This project is a simple **Rock Paper Scissors** game developed with Python.

The player competes against the computer. The computer randomly chooses an option and the winner is determined according to the game rules.

The goal of the game is to **reach 3 points and win**.

---

## ✨ Features

- 🎮 Player vs computer gameplay
- 🤖 Random computer choices
- 🏆 First player to reach 3 points wins
- 📊 Score tracking system
- ❌ Invalid input detection
- 🚪 Exit option

---

## 🕹️ Usage

Run the game with:

```bash
python rock_paper_scissors.py
```

After starting the game, choose one of:

```
rock
paper
scissors
```

Extra commands:

```
sc → Show score
q  → Quit the game
```

---

## 🎯 Game Rules

- 🪨 Rock beats Scissors
- 📄 Paper beats Rock
- ✂️ Scissors beats Paper

If both players choose the same option, the round ends in a draw.
