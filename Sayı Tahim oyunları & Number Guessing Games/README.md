# 🎯 Sayı Tahmin Oyunu

Python ile geliştirilmiş gelişmiş bir sayı tahmin oyunudur.

Oyuncunun amacı, bilgisayarın rastgele tuttuğu sayıyı tahmin etmektir. 
Ancak bu oyun klasik sayı tahmin oyunlarından farklı olarak birçok özel mod, zorluk seviyesi ve sürpriz mekanik içerir.

## 🚀 Özellikler

✅ Kolay, Normal, Zor ve Hardcore zorluk seviyeleri  
✅ Özel mod seçme sistemi  
✅ %55 ihtimalle rastgele Şans Modu sistemi  
✅ 19 farklı özel oyun modu  
✅ Tarih ve saat gösterimi  
✅ Hatalı giriş kontrolü  
✅ Deneme hakkı sistemi  
✅ Süre sınırı sistemi  
✅ Terminal tabanlı kullanıcı arayüzü  

## 🎮 Oyun Modları

### Ana Modlar

- 🟢 Kolay → 1-100 arası sayı
- 🟡 Normal → 1-200 arası sayı
- 🔴 Zor → 1-500 arası sayı
- 💀 Hardcore → 1-500 arası sayı, sadece 10 tahmin hakkı

## 🕹️ Nasıl Oynanır?

1. Oyunu çalıştırın.
2. Bir zorluk seviyesi seçin.
3. Bilgisayarın tuttuğu sayıyı tahmin etmeye çalışın.
4. Verilen ipuçlarını kullanarak doğru cevaba ulaşın.
5. Kazanmak için gizli sayıyı bulun.

# 🎯 Number Guessing Game

A powerful number guessing game developed with Python.

The player's goal is to guess the number randomly selected by the computer.
Unlike classic guessing games, this project includes multiple difficulty levels, special modes, and unique gameplay mechanics.

## 🚀 Features

✅ Easy, Normal, Hard and Hardcore difficulty levels  
✅ Custom mode selection system  
✅ 55% random Chance Mode system  
✅ 19 different special game modes  
✅ Date and time display  
✅ Input validation system  
✅ Limited attempts system  
✅ Time challenge system  
✅ Terminal-based interface  

## 🎮 Game Modes

### Main Modes

- 🟢 Easy → Number between 1-100
- 🟡 Normal → Number between 1-200
- 🔴 Hard → Number between 1-500
- 💀 Hardcore → Number between 1-500 with only 10 attempts
